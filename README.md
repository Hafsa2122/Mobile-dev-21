# Mobile Dev Studios Landing Page

A premium, high-performance landing page for a creative digital agency, built with React, Vite, Tailwind CSS, and Framer Motion. This project features stunning animations, a modern design system, and a fully responsive layout.

## ✨ Features

- **Dynamic Animations**: Seamless scroll interactions and reveal effects powered by Framer Motion.
- **Glassmorphism UI**: Modern aesthetic with blurred backdrops and vibrant gradients.
- **Micro-interactions**: Interactive buttons, hover states, and smooth navigation.
- **Fully Responsive**: Optimized for all screen sizes, from mobile devices to large desktop monitors.
- **Performance Optimized**: Built with Vite for ultra-fast development and optimized production builds.
- **Custom Design System**: Consistent typography, color palette, and spacing.

## 🛠️ Tech Stack

- **Framework**: [React 19](https://react.dev/)
- **Build Tool**: [Vite](https://vitejs.dev/)
- **Styling**: [Tailwind CSS 4](https://tailwindcss.com/)
- **Animations**: [Framer Motion](https://www.framer.com/motion/) & [GSAP](https://greensock.com/gsap/)
- **Icons**: [Lucide React](https://lucide.dev/)
- **Language**: [TypeScript](https://www.typescriptlang.org/)

## 🚀 Getting Started

### Prerequisites

- Node.js (Latest LTS version recommended)
- npm or yarn

### Installation

1. **Clone the repository:**
   ```bash
   git clone https://github.com/Hafsa2122/Mobile-dev-21.git
   cd Mobile-dev-21
   ```

2. **Install dependencies:**
   ```bash
   npm install
   ```

3. **Start the development server:**
   ```bash
   npm run dev
   ```

4. **Open in your browser:**
   Navigate to `http://localhost:5173`
